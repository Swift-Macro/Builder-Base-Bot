# Builder-Base-Bot
This is a standalone Builder Base macro made by Swift Macro. Only works for google play games pc. This bot quits an attack once it reaches 2 stars or after 45 seconds have passed. This was based on the video (https://www.youtube.com/watch?v=-YHt_uz3noI) made by BDLegend.

The bot will only work if the screen resolution is 1920 x 1080 and it is set as the primary monitor. Alternatively, you can take screenshots of the images and replace them with your own.
To use the bot, run the exe or install the required packages in a Python 3.10 virtual environment after cloning the repository. If you choose to clone the repository, run the following command in the command line: pip install -r requirements.txt
