# Builder-Base-Bot
This is a standalone Builder Base macro made by Swift Macro. It is designed for Google Play Games on PC.

The bot automatically stops an attack once it reaches 2 stars or after 45 seconds have passed. It is based on the video:
https://www.youtube.com/watch?v=-YHt_uz3noI by BDLegend.

## Requirements
- Screen resolution must be set to **1920x1080**
- Must be on the **primary monitor**
- If needed, you can replace the images in the `Images/BuilderBaseImages` folder with your own screenshots for better accuracy

## How to use

### Option 1 (Recommended)
Download and run the `.exe` file from the latest release.

### Option 2 (Python)
If you clone the repository, install dependencies with: 

pip install -r requirements.txt

Then run: python BuilderBaseGui.py
